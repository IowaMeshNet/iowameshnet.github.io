### Nano G2 Ultra
Intro:
A pocket-sized, efficient node with wideband antenna support, optimized for low-power operation.

Key Features:
• 	Nordic nRF52840 MCU
• 	SX1262 LoRa transceiver
• 	Wideband antenna (~815–940 MHz)
• 	Compact design with efficient power use

Considerations:
• 	Smaller battery capacity
• 	Limited display/interaction options
• 	Ideal for discreet, portable deployments
